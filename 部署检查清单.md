## 🚀 部署步骤

### 步骤 1：上传部署脚本到服务器

```bash
# 在本地 PowerShell 执行
scp deploy.sh root@你的服务器 IP:/root/
```

- [ ] 脚本已上传成功

### 步骤 2：SSH 登录服务器

```bash
ssh root@你的服务器 IP
```

- [ ] 登录成功

### 步骤 3：执行部署脚本

```bash
cd /root
chmod +x deploy.sh
sudo ./deploy.sh
```

按提示输入：
- [ ] Git 仓库地址
- [ ] SiliconFlow API Key

### 步骤 4：等待部署完成

脚本会自动执行以下步骤：
- [ ] 更新系统
- [ ] 安装 Node.js 18
- [ ] 安装 PM2、Nginx、Git
- [ ] 克隆项目代码
- [ ] 安装依赖
- [ ] 构建前端
- [ ] 配置环境变量
- [ ] 配置 Nginx 反向代理
- [ ] 启动应用
- [ ] 配置防火墙

### 步骤 5：安装 SSL 证书

```bash
apt install -y certbot python3-certbot-nginx
certbot --nginx -d koudai.xin -d www.koudai.xin
```

- [ ] SSL 证书已安装
- [ ] HTTPS 可以正常访问
