## 🚀 快速部署步骤

### 步骤 1：上传部署脚本到服务器

在本地 PowerShell 中执行：

```powershell
# 方法 1: 使用 scp (需要 Git Bash 或 WSL)
scp deploy-alibaba-cloud.sh root@47.116.122.157:/root/

# 方法 2: 使用 PowerShell 的 PSCP (需要下载 pscp.exe)
# 下载地址：https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html
.\pscp.exe deploy-alibaba-cloud.sh root@47.116.122.157:/root/

# 方法 3: 直接在服务器上下载（推荐）
# 先将脚本上传到 GitHub Gist 或其他可公开访问的位置
# 然后在服务器上执行：
# curl -O https://your-url/deploy-alibaba-cloud.sh
```

### 步骤 2：SSH 连接到服务器

```bash
ssh root@47.116.122.157
```

### 步骤 3：执行部署脚本

```bash
# 给脚本执行权限
chmod +x deploy-alibaba-cloud.sh

# 运行脚本
sudo ./deploy-alibaba-cloud.sh
```

脚本会提示你输入：
1. **Git 仓库地址** - 你的项目代码仓库 URL
2. **SiliconFlow API Key** - 用于 AI 服务（获取地址：https://cloud.siliconflow.cn/）

### 步骤 4：配置域名 DNS 解析

登录阿里云 DNS 控制台，添加以下解析记录：

| 记录类型 | 主机记录 | 记录值 | TTL |
|---------|---------|--------|-----|
| A | @ | 47.116.122.157 | 10 分钟 |
| A | www | 47.116.122.157 | 10 分钟 |

等待 5-10 分钟让 DNS 生效。

### 步骤 5：安装 SSL 证书（推荐）

```bash
# 安装 Certbot
apt install -y certbot python3-certbot-nginx

# 获取免费 SSL 证书
certbot --nginx -d koudai.xin -d www.koudai.xin
```

按照提示输入邮箱，同意条款后会自动配置 HTTPS。

## 🐛 故障排查

### 1. 后端服务无法启动

```bash
# 查看 PM2 日志
pm2 logs outlook-chat-backend

# 检查 .env 文件配置
cat /var/www/outlook-chat-plugin/.env

# 检查端口占用
netstat -tlnp | grep :3000

# 手动测试启动
cd /var/www/outlook-chat-plugin
node src/backend/server.js
```

### 2. AI 服务不工作

```bash
# 检查 API Key 是否正确
cat /var/www/outlook-chat-plugin/.env | grep SILICONFLOW

# 重启服务
pm2 restart outlook-chat-backend

# 查看 AI 相关日志
pm2 logs outlook-chat-backend | grep -i "AI"
```

### 3. Nginx 502 错误

```bash
# 检查后端是否运行
pm2 list

# 检查端口监听
netstat -tlnp | grep 3000

# 查看 Nginx 错误日志
tail -f /var/log/nginx/koudai.xin-error.log

# 测试后端直接访问
curl http://localhost:3000/api/chat
```

### 4. 页面 404

```bash
# 检查静态文件是否存在
ls -la /var/www/outlook-chat-plugin/dist/

# 检查 Nginx 配置
cat /etc/nginx/sites-available/koudai.xin

# 检查文件权限
chown -R www-data:www-data /var/www/outlook-chat-plugin/
chmod -R 755 /var/www/outlook-chat-plugin/
```

### 5. 域名无法访问

```bash
# 检查 DNS 是否生效
ping koudai.xin
nslookup koudai.xin

# 检查 Nginx 是否运行
systemctl status nginx

# 检查防火墙
ufw status
```

## 🔒 安全建议

### 1. 防火墙配置

```bash
# 只开放必要端口
ufw allow ssh
ufw allow http
ufw allow https
ufw enable
```

### 2. 定期更新

```bash
# 系统更新
apt update && apt upgrade -y

# Node.js 依赖更新
cd /var/www/outlook-chat-plugin
npm update
pm2 restart outlook-chat-backend
```

### 3. 监控日志

```bash
# 定期查看日志
pm2 logs --lines 100
tail -f /var/log/nginx/error.log

# 设置日志轮转（防止日志文件过大）
pm2 install pm2-logrotate
pm2 set pm2-logrotate:max_size 10M
pm2 set pm2-logrotate:retain 7
```
