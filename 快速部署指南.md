# 快速部署到阿里云

## 🚀 方法一：一键部署脚本（推荐）

### 步骤 1：上传部署脚本到服务器

```bash
# 在本地执行
scp deploy.sh root@你的服务器 IP:/root/
```

### 步骤 2：执行部署脚本

```bash
# SSH 登录服务器
ssh root@你的服务器 IP

# 给脚本执行权限
chmod +x deploy.sh

# 运行脚本
sudo ./deploy.sh
```

脚本会自动：
- ✅ 安装 Node.js、PM2、Nginx
- ✅ 克隆项目代码
- ✅ 安装依赖
- ✅ 配置环境变量
- ✅ 配置 Nginx
- ✅ 启动应用

### 步骤 3：配置域名

在阿里云 DNS 控制台添加解析：
- 记录类型：A
- 主机记录：@ 和 www
- 记录值：你的服务器 IP

### 步骤 4：安装 SSL 证书（推荐）

```bash
# 安装 Certbot
apt install -y certbot python3-certbot-nginx

# 获取证书
certbot --nginx -d koudai.xin -d www.koudai.xin
```

## 📦 方法二：手动部署

### 1. 连接服务器

```bash
ssh root@你的服务器 IP
```

### 2. 安装环境

```bash
# 更新系统
apt update && apt upgrade -y

# 安装 Node.js 18
curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs

# 安装 PM2
npm install -g pm2

# 安装 Nginx
apt install -y nginx

# 安装 Git
apt install -y git
```

### 3. 部署项目

```bash
# 创建目录
mkdir -p /var/www/outlook-chat-plugin
cd /var/www/outlook-chat-plugin

# 克隆代码
git clone 你的仓库地址 .

# 安装依赖
npm install

# 构建前端
npm run build
```

### 4. 配置环境变量

```bash
# 复制示例文件
cp .env.example .env

# 编辑文件
nano .env
```

填入：
```env
SILICONFLOW_API_KEY=sk-your-api-key-here
PORT=3000
```

### 5. 启动后端

```bash
# 启动应用
pm2 start src/backend/server.js --name outlook-chat-backend

# 保存进程
pm2 save

# 设置开机自启
pm2 startup | tail -1 | bash
```

### 6. 配置 Nginx

```bash
cat > /etc/nginx/sites-available/koudai.xin << 'EOF'
server {
    listen 80;
    server_name koudai.xin www.koudai.xin;

    location / {
        root /var/www/outlook-chat-plugin/dist;
        index chat.html;
        try_files $uri $uri/ /chat.html;
    }
    
    location /api/ {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
EOF

# 启用站点
ln -s /etc/nginx/sites-available/koudai.xin /etc/nginx/sites-enabled/
rm -f /etc/nginx/sites-enabled/default

# 重启 Nginx
nginx -t && systemctl restart nginx
```

### 7. 配置 SSL（可选但推荐）

```bash
apt install -y certbot python3-certbot-nginx
certbot --nginx -d koudai.xin -d www.koudai.xin
```



## 🐛 故障排查

### 1. 后端无法启动

```bash
# 查看日志
pm2 logs outlook-chat-backend

# 检查 .env 文件
cat /var/www/outlook-chat-plugin/.env

# 手动测试
cd /var/www/outlook-chat-plugin
node src/backend/server.js
```

### 2. API 返回错误

```bash
# 测试本地 API
curl http://localhost:3000/api/chat -d '{"messages": []}' -H "Content-Type: application/json"

# 查看后端日志
pm2 logs outlook-chat-backend --lines 100
```

### 3. Nginx 502 错误

```bash
# 检查后端是否运行
pm2 list

# 检查端口
netstat -tlnp | grep 3000

# 查看 Nginx 错误日志
tail -f /var/log/nginx/koudai.xin-error.log
```

- CPU: 1 核

- 带宽：1Mbps

- CPU: 2 核

- 带宽：3Mbps+
